# B-W_Master_Slave_RF
Design, validate, and release two manufacturable PCBs that together form a one-way wireless link: a solar-powered remote node (**R_Slave**) transmits a 433.92 MHz ASK/OOK signal to a mains-powered base unit (**Master_T**), which relays the received data to a host PC over USB. The end state is a complete KiCad design package — schematic, layout, BOM
